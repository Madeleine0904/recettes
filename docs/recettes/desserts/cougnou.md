## Préparation

- Dorure 1 jaune oeuf avec de eau ;
- Mode 2h20 dans la machine à pain ;
- Puis +/- 20 minutes au four 180-200 degrés C ;
- Mettre le jaune d'oeuf les 5 dernieres minutes ;
